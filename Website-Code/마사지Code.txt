              <h1>3.양쪽 귀 마사지</h1>

                  - 전체적인 피로를 풀어주고 에너지를 돋아준다.<br>    
              </h2>
           <h3>방법:<br>
            1.     아이가 아프지 않을 정도의 약한 압력으로

            2.     눈과 눈 사이의 부분을 아래에서 위로 쓸어주듯 만지면 
            스트레스 완화의 효과를 볼 수 있다.<br>
    
            3.     두 과정을 반복하여 주며 아이가 눈을 떴을 때에는 시선 교환을 하며 마무리한다.<br>
           </h3>
